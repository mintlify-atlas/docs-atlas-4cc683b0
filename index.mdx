title: "swift-syntax"

---

swift-syntax gives you a source-accurate, immutable syntax tree for any Swift source file. Use it to parse Swift code, build code generators, write Swift macros, implement refactoring actions, or create editor tooling — all without depending on a specific Swift toolchain version.

## Library overview

swift-syntax is a collection of focused, composable libraries. Most use cases only need a subset.

<Steps>
  <Step title="Parse Swift source">
    `SwiftParser` converts source text into a fully typed syntax tree with no toolchain dependency.
  </Step>
  <Step title="Inspect and traverse nodes">
    `SwiftSyntax` provides `SyntaxProtocol`, `SyntaxVisitor`, `SyntaxRewriter`, and hundreds of typed syntax node types.
  </Step>
  <Step title="Generate new code">
    `SwiftSyntaxBuilder` offers a result-builder DSL so you can construct valid Swift syntax nodes using familiar Swift syntax.
  </Step>
  <Step title="Write macros">
    `SwiftSyntaxMacros` defines the macro protocol hierarchy. `SwiftCompilerPlugin` wires your macro implementations into the Swift compiler.
  </Step>
</Steps>

<Note>
  swift-syntax releases are aligned with Swift language releases. For example, version `600.x.x` targets Swift 6.0. Always use the version that matches your Swift toolchain.
</Note>
